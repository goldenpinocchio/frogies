## Variant: Pastel Pond

### Design stance
A soft, collectible sticker-sheet look that makes Frogies feel cute, plush, and approachable.

### Key choices
- Layout: big featured hero with a supporting grid of sample Frogies
- Typography: rounded, friendly system font stack
- Color: pastel gradients, mint, lavender, sky blue, peach, and gold accents
- Interaction: filter pills plus click-to-promote frog cards

### Trade-offs
- Strong at: warmth, kawaii feel, easy visual scanning
- Weak at: less dramatic / less edgy for degen audiences

### Best for
- Collection preview pages
- Community-first brand presentation
- Sharing the first visual direction with the user
